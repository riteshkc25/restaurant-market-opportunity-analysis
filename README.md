# Overall
For this, I integrated three distinct data sources (Yelp, Census, Zillow). I engineered meaningful metrics like a 'value score' to identify quality-to-price outliers and 'opportunity score' to identify potential opportunity for marked entry and conducted geospatial analysis to map competitive dynamics. The result was a multi-dimensional view of the NYC restaurant landscape that revealed market gaps, consumer preferences, and investment opportunities.
